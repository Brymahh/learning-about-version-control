# Learning about Version Control

Author: Godsgift Braimah

Solving merge conflicts takes conscious effort, but I can do it! <br>
Frankly, what I enjoy about version control is the ability to experiment and make changes to my code, knowing I can always revert to a clean previous state. (Just incase I mess things up!)

The most frustrating would be handling merge conflicts, especially when several people have modified the same parts of a file, it can be confusing to untangle what should stay and what should be changed.
